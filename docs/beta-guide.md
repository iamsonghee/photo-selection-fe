## 1. 첫 프로젝트 만들기

[스크린샷 포함]

**목적:** 촬영 건별로 프로젝트를 생성해 고객 셀렉 작업을 관리합니다.

**단계:**

**1단계: 로그인**
- https://acut.vercel.app 에 접속합니다.
- "Google로 시작하기" 버튼을 클릭해 구글 계정으로 로그인합니다.
- ※ 베타 기간 중 구글 로그인만 지원합니다. 카카오 로그인은 정식 출시 후 제공 예정입니다.

**2단계: 대시보드 진입**
- 로그인하면 대시보드로 자동 이동합니다.

**3단계: 새 프로젝트 생성**
- 화면 오른쪽 하단의 **"새 프로젝트" (+) 버튼**을 클릭합니다.

**4단계: 프로젝트 정보 입력**

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 프로젝트명 | 필수 | 내부 관리용 이름 (고객에게 노출됨) |
| 촬영 유형 | 필수 | 웨딩 / 가족·베이비 / 졸업·기념 / 프로필·증명 / 기타 |
| 촬영 일자 | 필수 | 실제 촬영한 날짜 |
| 고객 이름 | 필수 | 고객 갤러리 화면에 표시됨 |
| 고객 연락처 | 선택 | 내부 메모용 |
| 셀렉 갯수 N | 필수 | 고객이 최종 선택해야 할 사진 수 |
| 셀렉 기한 | 필수 | 3 / 5 / 7 / 14 / 30일 중 선택 |
| 위치 | 선택 | 촬영 장소 |
| 고객 비밀번호 (PIN) | 선택 | 설정 시 링크를 받은 사람도 PIN 입력 필요 |
| 재보정 횟수 | 선택 | 0 / 1 / 2회 — 보정본 피드백 라운드 수 |

- 입력 완료 후 **"만들기" 버튼**을 클릭합니다.

**팁:**
- 셀렉 기한은 프로젝트 생성 후에도 수정할 수 있어요.
- PIN을 설정했다면 링크와 비밀번호를 함께 고객에게 전달해야 합니다.
- 베타 기간 중 프로젝트는 최대 **10개**까지 생성 가능합니다.

## 2. 사진 업로드하기

[스크린샷 포함]

**목적:** 촬영한 원본 사진을 업로드해 고객이 선택할 수 있는 갤러리를 만듭니다.

**단계:**

**1단계: 프로젝트 진입**
- 대시보드에서 해당 프로젝트 카드를 클릭합니다.

**2단계: 업로드 시작**
- 프로젝트 상세 페이지 상단의 **"원본 업로드" 버튼**을 클릭합니다.

**3단계: 파일 선택**
- 업로드할 사진 파일을 선택합니다. 단일 또는 다중 선택 모두 가능합니다.
- 지원 형식: **JPEG, PNG, WebP, HEIC/HEIF**

**4단계: 업로드 진행 확인**
- 업로드 진행률이 화면에 표시됩니다. 프로젝트 상태가 "업로드 중"으로 표시됩니다.

**5단계: 업로드 완료 확인**
- 셀렉 갯수(N장) 이상 업로드가 완료되면 화면 하단의 **"고객 초대 링크 활성화" 버튼**이 활성화됩니다.
- 다음 섹션에서 이 버튼으로 고객 초대를 진행합니다.

**팁:**
- RAW 파일(CR3, NEF 등)은 지원하지 않습니다. JPEG 또는 HEIC로 변환 후 업로드해 주세요.
- 한 프로젝트에 최대 **1,500장**까지 업로드 가능합니다 (베타 제한).
- 업로드 도중에 갤러리에서 실시간으로 진행 상황을 확인할 수 있어요.

## 3. 고객 초대 링크 생성 및 공유

[스크린샷 포함]

**목적:** 고객이 사진을 선택할 수 있도록 전용 갤러리 링크를 활성화하고 공유합니다.

**단계:**

**1단계: 초대 링크 활성화**
- 업로드 페이지 하단의 **"고객 초대 링크 활성화" 버튼**을 클릭합니다.
- 이 버튼은 셀렉 갯수(N장) 이상 업로드됐을 때만 클릭할 수 있어요.

**2단계: 공유 모달 확인**
- 버튼 클릭 즉시 프로젝트 상태가 **"셀렉 중"**으로 변경됩니다.
- 공유 모달이 자동으로 열립니다.

**3단계: 링크 복사**
- PIN을 설정한 경우: **"링크와 비밀번호 복사"** 버튼을 클릭합니다.
- PIN이 없는 경우: **"초대 링크 공유"** 버튼을 클릭합니다.

**4단계: 고객에게 전달**
- 복사된 링크를 카카오톡, 문자 등으로 고객에게 **직접 전달**합니다.
- ※ 앱 내 자동 전송 기능은 베타 기간 중 제공하지 않습니다.

**5단계: 고객 셀렉 시작**
- 고객이 링크에 접속하면 갤러리에서 사진 선택을 시작합니다.

**팁:**
- "링크와 비밀번호 복사" 버튼으로 링크와 PIN을 한 번에 클립보드에 복사할 수 있어요.
- 나중에 링크를 다시 공유해야 할 때는 프로젝트 페이지에서 언제든 다시 복사 가능합니다.
- 고객은 갤러리에서 사진마다 별점(1~5점)과 색상 태그를 달 수 있어요. 보정 참고 자료로 활용하세요.

## 4. 고객 셀렉 결과 확인하기

[스크린샷 포함]

**목적:** 고객이 최종 확정한 사진 목록을 확인하고 보정을 시작합니다.

**단계:**

**1단계: 셀렉 완료 확인**
- 대시보드에서 해당 프로젝트의 상태 배지가 **"셀렉 완료"**로 변경됐는지 확인합니다.

**2단계: 결과 페이지 진입**
- 프로젝트 카드를 클릭한 뒤, 워크플로우에서 **"셀렉 확인"** 단계를 클릭합니다.

**3단계: 선택 결과 확인**
- 고객이 선택한 사진 목록과 각 사진에 남긴 코멘트를 확인합니다.

**4단계: 목록 내보내기 (선택)**
- **"CSV"** 버튼: 파일명 목록을 CSV 파일로 다운로드합니다.
- **"TXT"** 버튼: 파일명 목록을 TXT 파일로 다운로드합니다.
- **"CLIPBOARD"** 버튼: 파일명 목록을 클립보드에 복사합니다.

**5단계: 보정 시작**
- 결과를 모두 확인했으면 **"보정 시작하기" 버튼**을 클릭합니다.
- 프로젝트 상태가 "보정 중"으로 전환됩니다.

**팁:**
- "보정 시작하기"를 클릭하면 고객은 더 이상 최종 확정을 취소할 수 없어요. 클릭 전에 선택 목록을 꼭 확인해 주세요.
- 고객이 코멘트를 남겼다면 결과 페이지에서 먼저 읽고 보정을 시작하세요.

## 5. 자주 묻는 질문 (FAQ)

**Q1. 고객이 링크를 받았는데 사진이 안 보인다고 해요.**

업로드가 완전히 완료됐는지 확인해 주세요. 업로드 중 상태에서는 갤러리에 사진이 보이지 않을 수 있습니다. 완료 후 링크를 다시 전달해 주세요.

---

**Q2. 셀렉 기한 내에 고객이 완료하지 못했어요.**

프로젝트 상세 페이지의 수정 화면에서 셀렉 기한을 연장할 수 있어요. 고객에게 기한이 연장됐음을 별도로 알려주세요.

---

**Q3. 카카오톡으로 공유받은 링크에서 구글 로그인이 안 돼요. (작가 본인 로그인 시)**

카카오톡 내 브라우저에서는 구글 로그인이 제한됩니다.

- **iOS**: 우측 하단 **···** 버튼 → "Safari로 열기"
- **Android**: 우측 상단 **···** 버튼 → "다른 앱으로 열기" → Chrome

※ 카카오 계정 로그인은 정식 출시 후 제공 예정입니다.

---

**Q4. 프로젝트를 몇 개까지 만들 수 있나요?**

베타 기간 중에는 최대 **10개**까지 가능합니다. 한도에 가까워지면 대시보드 우측 사용량 패널에 알림이 표시됩니다.

---

**Q5. 어떤 사진 파일 형식을 올릴 수 있나요?**

**JPEG, PNG, WebP, HEIC/HEIF**를 지원합니다. RAW 파일(CR3, NEF 등)은 지원하지 않으니, JPEG 또는 HEIC로 변환 후 업로드해 주세요.
